# To-Do List Manager

A simple command-line To-Do List application built using Python.

This project allows users to manage daily tasks by adding, viewing, removing, and searching for tasks.

## Features

- View all tasks
- Add new tasks
- Remove tasks
- Search for tasks
- Continuous menu system
- User input validation

## Python Concepts Practiced

While building this project, I practiced and applied:

- Variables and user input
- Conditional statements (`if`, `elif`, `else`)
- `while` loops for continuous program execution
- `for` loops for iterating through lists
- Lists and list manipulation
- List methods:
  - `.append()`
  - `.pop()`
- String methods:
  - `.lower()`
  - `.capitalize()`
  - `.strip()`
- String formatting using f-strings
- Case-insensitive searching
- Functions to organize and reuse code
- Boolean variables for tracking conditions
- Breaking a larger program into smaller functions
- Basic program structure and user interaction
